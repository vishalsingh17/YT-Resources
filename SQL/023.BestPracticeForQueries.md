
## Bad Practice
```sql
SELECT EMPLOYEE_ID, FIRST_NAME, LAST_NAME,  DEPARTMENT_ID, SALARY, HIRE_DATE FROM emp_details WHERE  DEPARTMENT_ID IN (20, 50, 100) ORDER BY  SALARY DESC LIMIT 5;
```

## Good Practice
```sql
SELECT 
    EMPLOYEE_ID,
    FIRST_NAME,
    LAST_NAME,
    DEPARTMENT_ID,
    SALARY,
    HIRE_DATE
FROM 
    emp_details
WHERE 
    DEPARTMENT_ID IN (20, 50, 100)
ORDER BY 
    SALARY DESC
LIMIT 5;
```
