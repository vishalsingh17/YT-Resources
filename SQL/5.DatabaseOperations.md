#### 1. To show the list of all databases
```sql
show databases;
```

#### 2. Create a new database
Syntax:
```sql
create database <database_name>;
```
Eg:
```sql
create database practice_db;
```

#### 3. Delete an existing database
Syntax:
```sql
drop database <database_name>;
```
Eg:
```sql
drop database practice_db;
```
